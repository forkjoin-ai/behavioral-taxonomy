# Behavioral Taxonomy Data

Parent: [@affectively/behavioral-taxonomy](../README.md)

This directory contains the raw JSON datasets and generated corpus-level Gnosis projections.

## Files

- `behavioralLoops.json`: canonical behavioral loop corpus.
- `emotions.json`: emotion taxonomy dataset.
- `biases.json`: cognitive bias dataset.
- `traits.json`: personality trait dataset.
- [gnosis-loops](./gnosis-loops/README.md): generated per-loop `.gg`, `.tla`, `.cfg`, and score manifest artifacts derived from `behavioralLoops.json`.
